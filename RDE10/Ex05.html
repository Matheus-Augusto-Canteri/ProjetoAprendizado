Ajax com Java / 2007

Ajax com Java mostra como você pode fazer seus aplicativos web Java serem mais rápidos nas respostas e mais dinâmicos ao incorporar as novas características Java, incluindo listas, arrastar e soltar e muito mais. Este livro demonstra muitos meios diferentes para incorporar Ajax, a partir da construção de JavaScript em seus aplicativos 'à mão' ao usar o novo Kit de ferramentas da Web do Google (GWT). E, é claro, você aprenderá como usar Ajax em seus aplicativos Struts, JSP e JSF. 'Ajax com Java' começa com uma introdução ao Ajax, ensinando a você como escrever alguns aplicativos básicos que usam JavaScript do lado do cliente para pedir informações de um servlet Java e exibi-lo sem recarregar a página toda. Também apresenta muitas estratégias para a comunicação entre o cliente e o servidor, incluindo o envio de dados brutos e o uso de XML ou JSON para enviar conjuntos de dados mais complexos. O livro, então, estendeu-se em diferentes abordagens para incorporar Ajax. Essas abordagens incluem: - As bibliotecas de JavaScript Prototype e script.aculo.us, as bibliotecas de Dojo e de Rico e DWR - Uso de Ajax para construir formulários de entrada e campos de sugestão - Criação em biblioteca de tag de Ajax e bibliotecas de tag de terceiro - Integração de Ajax em aplicativos de JavaServer Pages (JSP) - Uso de Ajax com Struts - Integração de Ajax em aplicativos JavaServer Faces (JSF) - Uso do GWT do Google, que oferece uma abordagem pura de Java para desenvolver aplicativos web: seus componentes do lado do cliente são escritos em Java e compilados em HTML e JavaScript 'Ajax com Java' ensina a você como construir aplicativos que sejam mais interativos, dinâmicos, empolgantes e divertidos para seus usuários. Caso você seja um desenvolvedor de Java e não tenha tentado o Ajax, porém, gostaria de começar, este livro é essencial.

AJAX : guia de programação / 2006

Para atender aos requisitos de usabilidade exigidos pelo mercado surge o Ajax (Asynchronous Javascript and XML) que oferece uma interface mais amigável e com respostas mais rápidas na web. Este livro explica as características e peculiaridades das tecnologias fundamentais do Ajax como HTML XHTML JavaScript DOM CSS XML XSLT e XMLHttpRequest com exemplos e algumas dicas de utilização. Analisa em detalhes uma miniaplicação novas metodologias para o desenvolvimento e gerenciamento de projetos web e frameworks para dar suporte às aplicações Ajax. É indicado a gerentes de projeto projetistas analistas programadores e profissionais da área.

Ajax, rich internet aplications e desenvolvimento web para programadores / 2008

A prática e exclusiva abordagem da Série do Desenvolvedor Deitel trata dos mais recentes temas relacionados ao desenvolvimento de aplicações Web. Este novo livro dos consagrados autores Paul J. Deitel e Harvey M. Deitel apresenta a tradicional abordagem live-code - criada pelos próprios autores - totalmente dirigida ao ensino do desenvolvimento de aplicações para Web 2.0. Com lições demonstradas por meio de 180 programas (e 18.000 linhas de código) previamente testados, o leitor encontra aqui um completo guia para o aprendizado da sintaxe, das descrições de código e de suas respectivas saídas para desenvolver as mais fortes, seguras e flexíveis aplicaçõesWeb 2.0. A obra tem início com uma introdução concisa sobre os padrões XHTML, CSS e JavaScript. Em seguida, traz lições sobre o desenvolvimento avançado do lado do cliente, apresentando padrões como DOM, XML, Ajax, JSON e outras tecnologias Rich Internet Applications. Por fim, são abordados os aspectos do lado do servidor, incluindo serviços Web, bancos de dados, PHP, Ruby on Rails, ASP .NET, JavaServer Faces e serviços Web. Quando chegar ao último capítulo, você terá aprendido tudo o que precisa saber para construir as melhores aplicações Web 2.0 para seu site.
