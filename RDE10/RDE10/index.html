Index para os exercícios atuais e futuros

Exercício 01

Escreva uma página html que apresente os dados pessoais do funcionário de uma empresa: Nome completo, RG, CPF, matrícula, endereço (rua, número, cep, bairro, cidade, estado), contato (telefone residencial, telefone celular, e-mail). A página também deve mostrar uma foto do funcionário. Você deve utilizar tags de parágrafo, título, ênfase, negrito e imagem.

Exercício 02

Crie uma página html que contenha as informações de 10 automóveis antigos bem como imagens de cada automóvel escolhido. Nesta mesma página devem conter links externos sobre os automóveis. Devem ser utilizadas tags de parágrafo, título e imagem, vídeo e/ou audio.

Exercício 03

A coordenação do Curso de Tecnologia em Análise e Desenvolvimento de Sistemas deseja disponibilizar em uma página html os nomes, e-mails e links para o currículo lattes dos professores vinculados ao curso. Escreva a página solicitada utilizando os tags para parágrafo, título, ênfase, negrito e outras necessárias.

Exercício 04

Elabore uma página para um restaurante que contenha a apresentação e o cardápio. O Cardápio deve conter o nome do produto, o preço e uma imagem. Os ingredientes devem ser apresentados quando clicar sobre o nome do produto.

Exercício 05

A biblioteca do Campus Irati necessita de uma página HTML para exibir os dados sobre os títulos sobre Desenvolvimento Web de seu acervo.

Exercício 06

Escreva uma página html para uma agenda de contatos da turma. Essa agenda deve possuir uma página inicial na qual são mostrados apenas a foto e o nome do aluno. Ao clicar sobre um nome desta agenda o usuário é direcionado para outra página na qual devem ser mostrados além da foto e do nome completo o e-mail, o telefone residencial e o telefone celular do aluno. Os nomes devem ser separados por ordem alfabética e no topo da página deve conter as letras dos nomes A, B, C, D entre outras, ao clicar em uma letra a página deve ser rolada até a lista de nomes da letra escolhida.

Exercício 07

Utilizando como referência o W3Schools crie as tabelas abaixo em uma página html.
i. Menu de operações.
ii. Informações sobre alimentos.
iii. Passagens aéreas.
iv. Quadro de horários do Campus Irati (preencha com as disciplinas!).
v. Demonstrativo de resultado do exercício (DRE).
