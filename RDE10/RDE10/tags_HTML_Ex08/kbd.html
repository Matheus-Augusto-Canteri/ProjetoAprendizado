A kbd tag é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Obs: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS (veja o exemplo abaixo).

Exemplo:

Press Ctrl + C to copy text (Windows).

Press Cmd + C to copy text (Mac OS).
