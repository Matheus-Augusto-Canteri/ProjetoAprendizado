Hiperlinks

Hiperlinks, geralmente chamados de links, são um conceito fundamental da Web. Ele serve para conectar ao texto informações complementares, que você pode ler naquele momento ou posteriormente e podem trazer mais referências para tornar o conteúdo mais rico, além de não deixar o texto tão longo e extenso. Ao clicar nele, pode-se abrir uma caixa de texto e continuar na página ou mesmo ir para algum outro endereço ou ser uma conexão direta com algum download de programa ou aplicativo, por exemplo.

Irei mostrar todos os tipos de hiperlinks e como usá-los de maneira correta!


Link HTML - Sintaxe

Pela sintaxe, você colocar o comando "a href="url">link text /a", com o maior e menor devidamente posicionado, assim, ele irá abrir o site em outra aba, irei fazer um teste, vou colocar o link da OPEX aqui para você clicar site One Piece Ex. Resumindo, onde está escrito "url" você deve colocar a URL do site que você quer que abra e o "text" é o nome de como você quer chamá-lo no seu site.

Link HTML - Com atributo de destino

Com atributo de destino, funciona quase da mesma maneira, você irá utilizar o comando "a href="url">link text /a", com o maior e menor devidamente posicionado, porém, com a adição "target="_blank", se "target="_blank" o link será aberto em uma nova janela ou guia do navegador, como o exemplo a seguir, se usarmos "target_=self" ele abrirá na mesma aba. Também temos as opções de utilizar o "rel=_external" que seria para indicar que o link é externo e o "hreflang="en"" para indicar a linguagem do site que será aberto. visite One Piece Ex!.

Link HTML - URLs absolutos x URLs relativos

As URLs absulotas são aquelas citadas acima, que vão para outro site, utilizando o "https://", as URLs relativas lhe enviam para a mesma página dentro do mesmo site, sem a parte do "https://www". O exemplo a seguir de URL absoluta One Piece Ex, agora a URL relativa seria assim mangá 1040.

Link HTML - Hiperlink interno

Links tem que lhe enviar para algum lugar, correto? existe o hiperlink interno, que vão lhe enviar para um lugar diferente da mesma página, para acessar uma certa parte que você deseja se caso o documento for extenso, a sintaxe seria "a href="#título">título./a". Como o exemplo a seguir título.

Link HTML - Hiperlink local

São os links que vão lhe enviar para outras páginas, dentro do mesmo site, a sintaxe seria "a href="RDE03-Ex01-Hans_Boron.html">site exemplo./a". Como o exemplo a seguir: site exemplo.

Link HTML - Hiperlink em imagens

Existem links que vão lhe mostrar imagens, ou seja, vão estar envolvidos a tag de imagem e hiperlink, a sintaxe seria "a href="imagem" "img src="imagem.jpg".

Link HTML - Hiperlink para Download

São links que vão apontar arquivos para realizar o download, a sintaxe seria "a href="arquivo.extensão" download="arquivo.extensão" type="tipo">Conteúdo. Como o exemplo a seguir: download.

Link HTML - Hiperlink para e-mail

São links que vão apontar para um e-mail, a sintaxe seria "a href="mailto:emailteste@gmail.com">email@gmail.com /a". Como o exemplo a seguir: contato@contato.com.br.
