Questionário

1. Em síntese, como se deu a evolução da internet?
Na guerra fria os Estados Unidos em divergência com a União Soviética, eles projetaram algo pra bater de frente com o Projeto Sputnik, o qual tinham nomeado de ARPANET que foi conhecido como a mãe da internet, que serviu como comunicação alternativa para transmitir informações entre eles.
2. Quais foram as principais invenções que proporcionaram o desenvolvimento da internet?
ARPANET, ARPA INTERNET, TCP/IP, Internet e WWW
3. Explique os protocolos http e https.
HTTP é o protocolo de comunicação que é usado pela internet ao transferir dados do computador do usuário até os servidores de hipermídia.
HTTPS seria uma cópia do HTTP só de uma forma mais segura, porque é o HTTP embrulhado em uma camada SSL a qual faz com que os dados sejam criptografados ao transferir e ainda verifique a autenticidade por meio de certificados digitais.
4. Qual a função do DNS e porque a necessidade de sua existência?
DNS são nada mais que os responsáveis por localizar números IP, traduzi-los para os links dos sites que buscamos nos navegadores da web.
5. Caso não existisse o protocolo TCP/IP quais o problemas aconteceriam?
Ao eliminar os protocolos de comunicação padronizados seria quase impossível ter uma rede de informações de alcance global como a internet nos dias de hoje.
6. Explique brevemente o protocolo TCP/IP
O protocolo TCP/IP consiste em ser um protocolo de troca de informações que busca aumentar ainda mais as conexões das redes diferentes de modo a padronizar essa comunicação.
7. O que é hipertexto?
Estes são os famosos links que digitamos nas barras de pesquisa dos navegadores para buscar e entrar em sites, cujo mesmo conectam as páginas da internet, estando elas interligadas ou não.
8. Defina HTML.
HTML é o bloco de construção mais premeditar da web, sendo HTML uma sigla para HyperText Markup Language que na tradução para o português seria Linguagem de Marcação de HiperTexto.
